Language / Runtime

Notes

Package Manager¶

OpenClaw uses pnpm 10.23.0 as its package manager. The monorepo is configured via pnpm-workspace.yaml with workspace packages under extensions/, packages/, and ui/.

Note

Dependencies listed in pnpm.patchedDependencies must use exact versions (no ^ or ~ ranges) to ensure patches apply correctly.

Note

Bun is supported as an alternative runtime for development and testing (bun <file.ts>, bunx <tool>). Node.js remains the primary production runtime.
